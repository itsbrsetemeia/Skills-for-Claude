# Programming Skills

This directory contains Skills focused on software development, software engineering, code quality, debugging, architecture, and technical decision-making.

The purpose of these Skills is to help developers build reliable, maintainable, secure, and scalable software systems.

Whether you're a student learning to code, an experienced developer, or a technical leader, these Skills are designed to support every stage of the software development lifecycle.

---

## Included Skills

### bug-hunter

A specialist in debugging, root-cause analysis, troubleshooting, and issue resolution.

Use this Skill when:

* Investigating bugs
* Analyzing error messages
* Debugging applications
* Reviewing logs
* Troubleshooting production incidents
* Diagnosing performance issues

Focus areas:

* Root-cause analysis
* Debugging strategies
* Error investigation
* Performance bottlenecks
* Production issues

---

### code-reviewer

A specialist in code review, software quality, maintainability, security, and best practices.

Use this Skill when:

* Reviewing source code
* Improving code quality
* Finding bugs
* Detecting security issues
* Improving performance
* Evaluating maintainability

Focus areas:

* Clean Code
* SOLID principles
* Security
* Performance
* Readability
* Best practices

---

### software-architect

A specialist in software architecture, system design, scalability, infrastructure, and technical planning.

Use this Skill when:

* Designing software systems
* Planning APIs
* Building SaaS platforms
* Designing microservices
* Evaluating architectural decisions
* Planning system scalability

Focus areas:

* System design
* Scalability
* Reliability
* Security
* Infrastructure
* Long-term maintainability

---

## Philosophy

Great software is not defined solely by working code.

Great software is:

* Reliable
* Maintainable
* Secure
* Scalable
* Understandable

These Skills encourage thoughtful engineering practices that prioritize long-term success over short-term solutions.

---

## Recommended Usage

Choose the Skill that best matches your objective:

| Goal                         | Recommended Skill  |
| ---------------------------- | ------------------ |
| Find and fix bugs            | bug-hunter         |
| Review and improve code      | code-reviewer      |
| Design software architecture | software-architect |

For complex projects, these Skills work best when used together.

Example workflow:

1. Use **software-architect** to design the system.
2. Implement the solution.
3. Use **code-reviewer** to improve code quality.
4. Use **bug-hunter** to investigate issues and production problems.
