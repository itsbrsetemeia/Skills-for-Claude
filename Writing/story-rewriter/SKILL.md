---

name: story-rewriter
description: Rewrites excerpts from stories, novels, comics, and narratives by adding emotion, atmosphere, visual impact, and character depth without altering events, character personalities, or the original plot.
---

# Objective

Transform simple narrative text into more vivid, emotional, and engaging versions.

The goal is not to change the story.

The goal is to improve how it is told.

# When to Use

Use this Skill when:

* The user provides story excerpts.
* The user asks to improve a scene.
* The user requests more emotion.
* The user requests stronger narrative impact.
* The user wants more professional writing.
* The user wants a more cinematic scene.

# When NOT to Use

Do not use this Skill when:

* The user is asking for new story ideas.
* The user wants to change the plot.
* The user is asking questions about the story.
* The user only wants grammar correction.

# Absolute Rules

1. Never alter important events.

2. Never create new characters.

3. Never remove existing characters.

4. Never change powers, abilities, or established worldbuilding details.

5. Never contradict information provided by the user.

6. Preserve the original emotional intent of the scene.

7. Preserve important dialogue.

# Writing Guidelines

## Atmosphere

Add:

* Physical sensations.
* Sounds.
* Smells.
* Temperature.
* Movement.
* Environmental details.

Make the reader feel present in the scene.

## Emotion

Amplify:

* Fear.
* Hope.
* Anger.
* Sadness.
* Tension.
* Joy.
* Determination.

Show emotions through actions, expressions, and thoughts.

Avoid directly stating emotions when they can be demonstrated instead.

## Pacing

Alternate between:

* Short sentences for impact.
* Longer sentences for immersion.

Control pacing as if directing a film.

## Descriptions

Replace generic descriptions with memorable ones.

Example:

Weak:

"The hallway was dark."

Better:

"The hallway seemed to swallow the light, leaving only silent shadows dancing across the walls."

## Characters

Strengthen:

* Body language.
* Reactions.
* Thoughts.
* Internal conflicts.

Make characters feel alive.

## Dialogue

Preserve the original meaning.

But make dialogue:

* More natural.
* More expressive.
* More emotionally charged.

# Style

Blend:

* Modern clarity.
* Cinematic storytelling.
* Emotional impact.
* Smooth readability.

Avoid:

* Excessively ornate language.
* Overusing metaphors in every paragraph.
* Long descriptions that slow down the scene.

The writing should feel professional while remaining enjoyable to read.

# Intensity Levels

If the scene is calm:

* Low intensity.

If the scene is emotional:

* Medium intensity.

If the scene is dramatic:

* High intensity.

If the scene is epic:

* Maximum intensity.

# Response Format

Provide only:

1. Original text (optional if requested).
2. Rewritten text.

No explanations.

No comments.

No analysis.

# Example

Input:

"Thiago entered the room. He was nervous. Everyone looked at him."

Output:

"Thiago pushed the door open slowly. His heart pounded so hard it seemed to echo through the silent room. The moment he stepped inside, dozens of eyes turned toward him at once. For a brief instant, the air itself felt heavier."
