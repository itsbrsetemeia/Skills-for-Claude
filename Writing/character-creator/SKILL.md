# Objective

Create unique, compelling, and multifaceted characters that feel like real individuals, whether they are humans, aliens, gods, monsters, heroes, or villains.

The goal is not merely to create an appearance or a power.

The goal is to create complete individuals.

# When to Use

Use this Skill when:

* The user wants to create a character.
* The user wants to create a hero.
* The user wants to create a villain.
* The user wants to create an anti-hero.
* The user wants to create a team.
* The user wants to create supporting characters.
* The user wants to deepen existing characters.

# When NOT to Use

Do not use this Skill when:

* The user only wants a simple physical description.
* The user only wants grammar correction.
* The user is asking for real-world information.
* The user only wants a basic character sheet without depth.

# Character Creation Philosophy

Every character should feel like a person with a life of their own.

No character should exist solely to fulfill a plot function.

Every character should possess:

* Desires.
* Fears.
* Dreams.
* Flaws.
* Contradictions.
* Virtues.
* Traumas.
* Goals.

# Core Structure

## Identity

Define:

* Name.
* Nicknames.
* Titles.
* Age.
* Species.
* Nationality.
* Occupation.

## Appearance

Develop:

* Height.
* Physical build.
* Distinctive features.
* Clothing style.
* Characteristic expressions.
* Marks or scars.

Appearance should reflect the character's personality and history.

## Personality

Build:

* Strengths.
* Flaws.
* Virtues.
* Vices.
* Habits.
* Fears.
* Ambitions.

Avoid perfect characters.

# Psychology

Explore:

* How the character thinks.
* How they react under pressure.
* How they respond to failure.
* How they handle power.
* How they deal with loss.

# Backstory

Create:

* Origin.
* Family.
* Childhood.
* Defining events.
* Traumas.
* Achievements.
* Defeats.

The past should always influence the present.

# Motivation

Answer:

* What do they want?
* Why do they want it?
* What are they willing to sacrifice?
* What would they never do?

# Relationships

Develop:

* Friends.
* Rivals.
* Mentors.
* Allies.
* Enemies.
* Family.

No character exists in isolation.

# Heroes

When creating heroes:

* Define values.
* Define responsibilities.
* Define internal conflicts.
* Define human flaws.

Heroism should come with a cost.

# Villains

When creating villains:

* Avoid evil without purpose.
* Build a personal logic.
* Create understandable goals.
* Develop a personal philosophy.

The best villains genuinely believe they are right.

# Powers

When applicable:

## Origin

* Scientific.
* Supernatural.
* Technological.
* Cosmic.
* Biological.

## Limitations

Every power should have:

* Costs.
* Weaknesses.
* Consequences.
* Limits.

## Evolution

Define:

* Maximum potential.
* Advanced forms.
* Possible transformations.

# Memorable Characters

Aim to create:

* Unique traits.
* A distinct voice.
* Recognizable behaviors.
* Clear objectives.

Avoid generic archetypes without depth.

# Consistency

Continuously check for:

* Contradictions.
* Inconsistencies.
* Weak motivations.
* Unexplained changes.

Every action should make sense for the character.

# Response Format

When appropriate, present:

1. Identity
2. Appearance
3. Personality
4. Backstory
5. Motivation
6. Relationships
7. Powers
8. Trivia

Organize content clearly and professionally.

# Language Adaptation

Always respond in the same language used by the user.

Examples:

* If the user writes in Portuguese, respond in Portuguese.
* If the user writes in English, respond in English.
* If the user writes in Spanish, respond in Spanish.

# Supreme Rule

Do not simply create a character.

Create someone who feels as though they lived a life before the story began and will continue to exist after it ends.
